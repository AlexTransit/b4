# Веб-интерфейс

Веб-интерфейс b4 доступен по адресу `http://<IP>:7000` (порт по умолчанию). Через него выполняется всё управление: от настройки обхода до мониторинга соединений.

## Разделы

Боковое меню содержит следующие разделы:

- [Главная (Дашборд)](./dashboard) — состояние системы, метрики, активность устройств
- [Сеты](./sets/) — наборы настроек обхода
- [Дискавери](./discovery) — автоматический подбор конфигурации
- [DPI Детектор](./detector) — определение методов блокировки провайдера
- [Соединения](./connections) — мониторинг трафика в реальном времени
- [Логи](./logs) — системные логи
- [Настройки](./settings/) — параметры системы, сети, устройств, бэкап
